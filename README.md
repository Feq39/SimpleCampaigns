# SimpleCampaigns

## Requirements

- Java 25
- Node.js 18+
- npm

## Run backend

Windows:

```powershell
cd campains
.\mvnw.cmd spring-boot:run
```

Backend: `http://localhost:8080`

## Run frontend

In a second terminal:

```bash
cd frontend
npm install
npm run dev
```

Frontend: `http://localhost:5173`
